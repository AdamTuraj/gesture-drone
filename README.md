# Gesture Controlled Drone

![Drone Frame](https://cdn.hackclub.com/019d63e5-b5b9-79d6-85c3-c445e5284676/paste-1775497494746.png)

## Overview

This is my custom gesture-controlled micro drone project. The repo includes the frame and mechanical design, the ESP32-CAM board work, and software used for video streaming/control.

Still very much a work-in-progress on the software side.

## What's in this repo

- [CAD](CAD/) - Drone frame, mounts, props, and assembly files
- [PCB](PCB/) - Custom camera/ESP32 board design files, outputs, and BOM
- [Code](Code/) - Firmware and software for the drone and RPI

## Current Status

- Airframe and PCB design are done
- Firmware/software is in progress
- Iterating on full gesture control workflow
